## 关于
OsuMusic 是一个基于 Kotlin Multiplatform 打造的铺面音乐播放器。

## 功能
- 类 Osu!Lazer 的 UI 设计
- 铺面搜索和播放
- 击打音效（开发中）
- 背景视频/故事板（开发中）
- Osu! 个人资料卡（开发中）
- 播放列表（开发中）

## 构建
要构建和运行应用，你需要：
- 配置 **Kotlin Multiplatform** 开发环境。
- **Android Studio** 用于安卓开发。
- **Xcode** 用于 iOS 开发（如果目标平台是 iOS）。
### 安卓
直接在 Android Studio 中构建并运行。
### iOS
1. 在 Xcode 中更改 Bundle ID。
2. 构建并运行。

## 安装（对于 iOS）
使用 AltStore 或其他侧载工具安装 ipa 文件。
